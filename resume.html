* Hybrid mobile applications

Hybrid is a 'blend' of native and webapplication components. The core or the heart of the application is written with HTML, CSS and JavaScript. Code used for webapplications or development. So you open it with a browser I hear you think? Yes and no. Internally it opens with an embedded browser so is essentialy invisible to the end-user. It opens like a native app you download from the App-store or Google Play. A great advantage of hybrid apps is the cross-platform capability. It works on iOS, Android and for the limited number of fans: the Windows-phones! So no more coding in different sets of programming languages.

After the hands-on, intense training of 6 months, we where able to use and understand the principles of webdevelopment, basic object oriented programming and mySQL. We also learned to work with different frameworks like Bootstrap, Ionic and Angular (Typescript based). In my current function, I mainly work with Angular.
